##  Descripción del Proyecto
Este es un programa de consola desarrollado en **C#** que implementa una agenda telefónica modular. Permite interactuar con un menú de opciones mediante estructuras de control (`switch/case`) y funciones personalizadas.

##  Estructura del Código
* **Clase Principal:** `Program`
* **Método de Entrada:** `Main(string[] args)`
* **Métodos Auxiliares:** `PresioneContinuar()` para pausar la pantalla y limpiar flujos de lectura (`Console.ReadKey()`).

## Requisitos
* .NET SDK 
* IDE: Visual Studio Code o PyCharm / JetBrains Rider
